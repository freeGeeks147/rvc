# Ahmedabad 2030 Commonwealth Games Bid Website

A modern, responsive website for Ahmedabad's bid to host the 2030 Commonwealth Games.

## Features

- **Modern Design**: Clean, professional design with smooth animations
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive**: Engaging animations, hover effects, and real-time updates
- **Accessible**: Built with accessibility best practices
- **Performance**: Optimized for fast loading and smooth user experience

## Sections

1. **Hero Section**: Compelling introduction with call-to-action buttons
2. **About**: Why Ahmedabad is the perfect choice for the Games
3. **Sports**: Comprehensive sports program showcase
4. **Venues**: World-class venue information and capacity details
5. **Legacy**: Long-term benefits and impact for the community
6. **Support**: Interactive support form and real-time supporter counter

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **Real-time**: Socket.io for live updates
- **Styling**: Custom CSS with modern design principles
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone or download the project
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Website

1. Start the server:
   ```bash
   npm start
   ```

2. Open your browser and visit:
   ```
   http://localhost:3000
   ```

### Alternative Port

If port 3000 is in use, you can specify a different port:
```bash
PORT=3001 npm start
```

## API Endpoints

- `GET /` - Main website
- `POST /api/support` - Submit support form
- `GET /api/stats` - Get visitor statistics
- `GET /health` - Health check endpoint

The website is ready for deployment on any Node.js hosting platform such as:
- Heroku
- Vercel
- Netlify
- DigitalOcean
- AWS
- Hostinger (as requested)

### Hostinger Deployment

1. Upload all files to your Hostinger hosting account
2. Ensure Node.js is enabled in your hosting panel
3. Set the start command to `npm start`
4. Configure environment variables if needed
5. Your website will be live at your domain
